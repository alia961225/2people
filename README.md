# 2people
雙人十點半
遊戲目標：玩家的目標是將手中的牌點數總和控制在最接近或等於10.5，同時盡量贏過莊家。
遊戲規則：1.遊戲開始時，每位玩家和莊家都會先抽取一張牌。
  2.玩家可以選擇是否繼續抽取額外的牌，以逼近或達到10.5點。
  3.玩家在選擇停止抽牌時，進入比較階段。
  4.若玩家的點數超過10.5，即爆牌，該玩家輸掉該局。
  5.若莊家的點數超過10.5，即爆牌，則玩家獲勝。
  6.比較玩家和莊家的點數，點數最接近10.5的為贏家。
  7.若玩家達到五張牌且點數正好為10.5，或是直接抽到五張牌點數小於10.5，玩家立即獲勝。
  8.完成一局後，玩家可以選擇是否再次進行新的一局。
